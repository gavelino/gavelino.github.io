## Course Overview

This course introduces the principles and practice of Object-Oriented Programming (OOP) using the Java programming language. Students will:

- Understand the four pillars of OOP: encapsulation, inheritance, polymorphism, and abstraction
- Design and implement well-structured Java programs
- Apply object-oriented design patterns to solve common software engineering problems
- Write unit tests using JUnit to verify the correctness of their code
- Work with version control (Git) and collaborative development workflows

## Prerequisites

- Introduction to Programming (knowledge of variables, loops, and conditionals in any language)
- Basic understanding of algorithms and data structures

## Textbooks

- "Effective Java" by Joshua Bloch (3rd edition)
- "Head First Design Patterns" by Freeman & Robson
